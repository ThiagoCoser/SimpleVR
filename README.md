# SimpleVR

<img src="info.jpg" style='height: 100%; width: 100%; object-fit: contain'/>

SimpleVR é um projeto acadêmico, foi feito na Unity, para ser utilizado com óculos Stereoscópicos simples. O projeto foi construído sem assets prontos de XR, sendo criado com duas câmeras e desenvolvimento de scripts e lógica própria, com sentido de estudo e simplicidade.

### Features
- Rotação de cameras
- Uso de controle bluetooth
- Reset de cena
- Coleta de objetos

### Updates futuros
- Uso de giroscópio
